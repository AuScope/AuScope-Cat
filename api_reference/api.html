src.auscopecat.api.download(obj: SimpleNamespace, download_type: DownloadType, bbox: dict | None = None, srs_name: str = 'EPSG:4326', max_features=None, version='1.1.0', file_name: str = 'download.csv') → any[source]

Downloads data from object

Parameters:

  • obj – SimpleNamespace objects with “url”, “type” and “name” attributes

  • download_type – type of download

  • bbox – the bounding box for the download data e.g. {“north”:-31.456, “east”:129.653…}

  • srs_name – the SRS name, e.g. “EPGS:4326” (Optional)

  • max_features – maximum number of features to return (Optional)

  • version – WFS version, e.g. “1.1.0” (Optional)

  • file_name – the file name for the download (Optional)

Returns:
