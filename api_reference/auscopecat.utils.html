auscopecat.utils.validate_bbox(bbox: dict, adjust_bounds=False) → BoundingBox[source]

Validate a bounding box :param bbox: the bounding box, a dict with “north”, “south”, “east” and

“west” keys

Parameters:

adjust_bounds – if True, and the bounds are outside of the allowable range (i.e. north > 90.0, south < -90.0, east > 180.0 and west < -180.0) then an Exception will not be thrown but instead the values will be corrected to the closest lowest/highest value for that coordinate

:return the validated bounding box provided no AuScopeCatError is

thrown
