auscopecat.api module

Python library for accessing the AuScope Portal’s API methods.

auscopecat.api.search(pattern: str, ogc_types: list[ServiceType | str] | None = None, spatial_search_type: SpatialSearchType | str | None = None, bbox: dict | None = None, polygon: list[list[float]] | None = None) → list[SimpleNamespace][source]

Searches catalogue for service online resource results

Parameters:

  • pattern – search string. If there are multiple words, any result that contains any one of the words will be considered a match. To match an exact series of words, use quotation marks, e.g. “Broken Hill”.

  • ogc_types – limit results to those containing one or more of the specified OGC service types. This can be a list of ServiceTypes or strings. (Optional)

  • spatial_search_type – the type of spatial search (intersects, coontains within). Used with bbox and polygon. This can be a SpatialSearchType or string. (Optional)

  • bbox – the bounding box for the search data e.g. {“north”:-31.456, “east”:129.653…}

  • polygon – a list of points defined as a 2 element list of [latitude, longitude], e.g. [[-31.0, 125.0], [-32, 128.0], [-31.0, 128.0], [-31.0, 125.0]] (Optional)

Returns:

a list of SimpleNamespace objects of the form: namespace(url, type, name)

Searches catalogue for records

  • ogc_type – search for a certain kind of OGC service. This can be a list of ServiceTypes or strings. (Optional)

  • spatial_search_type – the type of spatial search (intersects, coontains within). Used with bbox and polygon. This can be a SpatialSearchType or string. (Optional)

  • bbox – the bounding box for the search data e.g. {“north”:-31.456, “east”:129.653…}

  • polygon – a list of points defined as a 2 element list of [latitude, longitude], e.g. [[-31.0, 125.0], [-32, 128.0], [-31.0, 128.0], [-31.0, 125.0]] (Optional)

auscopecat.api.download(obj: SimpleNamespace, download_type: DownloadType | str, bbox: dict | None = None, srs_name: str = 'EPSG:4326', max_features: int | None = None, version: str = '1.1.0', file_name: str | None = None) → any[source]

Downloads data from object

Parameters:

  • obj – SimpleNamespace objects with “url”, “type” and “name” attributes

  • download_type – type of download. This can be a DownloadType or string.

  • bbox – the bounding box for the download data e.g. {“north”:-31.456, “east”:129.653…}

  • srs_name – the SRS name, e.g. “EPGS:4326” (Optional)

  • max_features – maximum number of features to return (Optional)

  • version – WFS version, e.g. “1.1.0” (Optional)

  • file_name – the file name for the download (Optional)

Returns:

CSV data
